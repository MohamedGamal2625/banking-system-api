# banking-system-api
Tech Stack

- Java 17 – Core programming language for the API.
- Spring Boot – Framework for building RESTful APIs efficiently.
- Spring Data JPA – For database access and ORM.
- MySQL – Relational database to store customers and account data.
- Hibernate – JPA implementation for managing entities and relationships.
- Lombok – Reduces boilerplate code with annotations.
- Maven – Project and dependency management.
- Jakarta Validation – For input validation on entities.
